# Weather App (Flutter)

A cross-platform weather application built with Flutter that shows current weather based on device's location.

Features
----------------
* Current weather for your location
* Requesting data from API
* Location detection (GPS)
* Internet connection checking
* Offline caching of most recent data
* Changing temperature units: Celsius / Fahrenheit
* Animated UI

Key technologies
----------------
* Flutter
* bloc / flutter_bloc / hydrated_bloc
* http
* geocoder
* geolocator
* connectivity_plus
